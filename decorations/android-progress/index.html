Android Progress

What is this?

Android Progress is a series of developer-oriented humor posters featuring Bugdroid. All of them were (probably) designed by Andrew Bell and have references towards 1984-inspired propoganda.

View main description

Set 1

The Code For Victory print is of a green Bugdroid standing behind flying planes, encouraging the reader to code on.

The Comment That Code print has a Bugdroid worried about code that went undocumented.

The Feature Creep Kills print features a green Bugdroid working and a project...and the red feature creep.

The Design Develop Deploy print is a blue poster with an eye-catching red Bugdroid, following the Design Develop Deploy protocol.

Mini Set 1 includes all of these posters at small sizes.

Set 2

The Encrypt Your Bits print is of an unsuspecting Bugdroid getting spied on by a red Bugdroid hacker.

The Build It! print is of an Bugdroid helping with construction.

The Secure Connections print features a Bugdroid connecting a padlock with two red cords.

The Global Variable Conflict print has an evil-looking red Bugdroid behind globe.

Mini Set 2 includes all of these posters at small sizes.

Set 3

The Keep It Clean print has an Iconic-styled Bugdroid wiping away the 'spaghetti' code.

Invalidate Improper Input! is print of a Bugdroid fending off from a snake escaping from a laptop.

The Check out to Check In! is blue and orange poster of an Iconic-styled Bugdroid going through holes

The Back It Up! print features am orange poster with an Iconic-styled Bugdroid rushing to a device that has not been backed up.

Mini Set 3 includes all of these posters at small sizes.

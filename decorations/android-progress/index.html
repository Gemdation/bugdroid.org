Android Progress

Worker! Did you know Android Progress is a series of developed-oriented humor posters, featuring Bugdroid!?!? All of them were (probably) designed by Andrew Bell and have references towards propoganda seen likely inspired by 1984.

Set 1

The Code For Victory print is of a green Bugdroid standing behind flying planes, encouraging the reader to code on.

The Comment That Code print has a Bugdroid worried about code that went undocumented.

The Feature Creep Kills print features a green Bugdroid working and a project...and the red feature creep.

The Design Develop Deploy print is a blue poster with an eye-catching red Bugdroid, following the Design Develop Deploy protocol.

Set 2

The Encrypt Your Bits print is of an unsuspecting Bugdroid getting spied on by a red Bugdroid hacker.

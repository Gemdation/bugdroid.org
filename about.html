About

Everything Droid!

This is a Bugdroid fan site but it is first an archive of Bugdroid merchandise. But that doesn't mean it is limited to official merchandise, it also includes fan works too like the games!

Preservation

This website wouldn't be possible without the Internet Archive's help in easy access to these sites, so I'm trying to make this site easier to pick through as well.

I'm putting the merchandise's serial number in the meta keywords for search engines that decide to use it. If you are browsing on a computer and you view the source code of a page you'll see it!

'View' buttons

Throughout the products page, you will come across a product that is still up for sale. I have put links to the product pages where you still view and get it.

Droid Pack is not an affiliate of any retailer and always uses public web store links.

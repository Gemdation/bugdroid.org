About

The Standard Edition figure is classic Bugdroid with poseable arms, images are from the 2014 re-release. This was the first one of Series 1, and the first one entirely.

Contains

  • One figure
  • One (window) box
  • One carrier box (large quantity orders)

This item was exclusively on the Dead Zebra Shop and had since been taken off.

In 2014 it was redesigned to due to the new Android wordmark, the color stayed the same.

The original wordmark is on the window box release, while the 2014 refresh is a standard figurine box.
